# 🤖 Group Join Application Bot

Сучасний Telegram-бот для прийому заявок на вступ у закриту групу.

---

## ✨ Функції

- **5-крокова анкета** з прогрес-баром та можливістю редагувати будь-яке поле
- **Верифікація контакту** — тільки через кнопку, ручне введення заблоковано
- **Захист від обходу** — кожен крок валідується, текст між кроками видаляється
- **Кілька адмінів** — усі отримують повідомлення, всі можуть схвалити/відхилити
- **Синхронізація статусу** — після рішення одного адміна, у всіх оновлюється статус
- **Одноразове запрошення** — після схвалення генерується унікальне посилання для конкретного юзера
- **Адмін-панель** (`/admin`) — статистика та список заявок у реальному часі

---

## ⚙️ Налаштування

1. **Встановіть залежності:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Відкрийте `bot.py` і заповніть:**

   ```python
   BOT_TOKEN = "ВАШ_ТОКЕН_ВІД_BOTFATHER"
   GROUP_CHAT_ID = -1001234567890   # ID вашої групи
   GROUP_NAME = "Назва Групи"

   ADMIN_IDS: list[int] = [
       123456789,   # Telegram ID адміна 1
       987654321,   # Telegram ID адміна 2
   ]
   ```

3. **Запустіть:**
   ```bash
   python bot.py
   ```

---

## 📌 Як отримати Group Chat ID

1. Додайте `@userinfobot` у свою групу
2. Напишіть будь-що в групі — бот поверне ID (від'ємне число)
3. Також можна: додайте бота до групи, надайте права адміна

---

## 📌 Як отримати свій Telegram ID

Напишіть `/start` боту `@userinfobot` — він покаже ваш ID.

---

## 🔑 Права бота в групі

Бот повинен мати такі права в групі:
- ✅ Запрошувати користувачів за посиланням
- ✅ Адміністратор (для генерації invite link)

## 🔄 Флоу заявки

```
/start
  └─ [📝 Надіслати заявку]
       └─ Крок 1: Ім'я
            └─ Крок 2: Прізвище
                 └─ Крок 3: Місто/Село
                      └─ Review (перевірка + редагування)
                           └─ Крок 4: Контакт (тільки кнопка)
                                └─ Крок 5: Підтвердження → Надіслати
                                     └─ Адміни отримують заявку
                                          ├─ [✅ Схвалити] → юзер отримує invite link
                                          └─ [❌ Відхилити] → юзер отримує відмову
```

## 🛡️ Безпека

| Загроза | Захист |
|---------|--------|
| Ручний номер телефону | Блокується, видаляється, виводиться попередження |
| Чужий контакт | Перевірка `contact.user_id == user.id` |
| Пропуск кроків | ConversationHandler — неможливо |
| Текст під час Review | Видаляється автоматично |
| Несанкціоновані callback | Перевірка `is_admin()` |
| Подвійне схвалення | Перевірка `app["status"] != "pending"` |
